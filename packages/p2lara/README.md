# p2lara

**Decentralized LoraWan network based off of Ding.js**


```
npm install p2lara --save
```  

## Development

Develop using docker-compose:


```
docker-compose up
```

Rebuild container:


```
docker-compose up --build
```

Without docker:

```
npm install
npm run dev
```
